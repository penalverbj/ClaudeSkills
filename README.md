# ClaudeSkills

These are some useful skills I've made and use for my personal projects. Feel free to use as you please!

## Use

Download the .skill zips and add them to your .claude/skills folder. 

I added the extracted folders too so you can see what files are in the .skill zips before downloading them.

## Current Skills

### wcag-a11y
This is a skill that makes Claude build UI elements and UX with accessibility in mind. Things like support for keyboard navigation, text-to-speech assistants, legible fonts and colors, contrast, and other accessibility guidelines. 

More information can be found here: https://www.w3.org/WAI/standards-guidelines/wcag/ 

### secure-ui
A skill to make UI security a priority for Claude Code. Includes not logging sensitive information, validating and sanitizing input, managing local storage and other dev tools don't store sensistive info, etc. 


### typescript-best-practices
Exactly what is sounds like. Makes sure generated TypeScript code compiles cleanly under ```"strict": true.```
