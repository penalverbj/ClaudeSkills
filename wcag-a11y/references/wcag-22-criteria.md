# WCAG 2.2 — All 87 Success Criteria

Source: https://www.w3.org/TR/WCAG22/  
Levels: A (minimum) | AA (legal standard) | AAA (enhanced)  
★ = New in WCAG 2.2

## Principle 2: Operable

### 2.1 Keyboard Accessible
| # | Criterion | Level |
|---|---|---|
| 2.1.1 | Keyboard — all functionality available via keyboard | A |
| 2.1.2 | No Keyboard Trap — keyboard focus can always move away | A |
| 2.1.3 | Keyboard (No Exception) — all functionality via keyboard, no exceptions | AAA |
| 2.1.4 | Character Key Shortcuts — letter-only shortcuts can be disabled/remapped | A |

### 2.2 Enough Time
| # | Criterion | Level |
|---|---|---|
| 2.2.1 | Timing Adjustable — time limits can be extended or disabled | A |
| 2.2.2 | Pause, Stop, Hide — moving/blinking content can be paused | A |
| 2.2.3 | No Timing — timing is not essential | AAA |
| 2.2.4 | Interruptions — interruptions can be postponed | AAA |
| 2.2.5 | Re-authenticating — data preserved after re-authentication | AAA |
| 2.2.6 | Timeouts — users warned about data loss from inactivity | AAA |

### 2.3 Seizures and Physical Reactions
| # | Criterion | Level |
|---|---|---|

| 2.3.2 | Three Flashes — no content flashes more than 3 times/sec at all | AAA |

| 2.4.7 | Focus Visible — keyboard focus indicator is visible | AA |
| 2.4.8 | Location — info about user's location within site | AAA |
| 2.4.9 | Link Purpose (Link Only) — link purpose determinable from link text alone | AAA |
| 2.4.10 | Section Headings — section headings used to organize content | AAA |
| 2.4.11 ★ | Focus Not Obscured (Minimum) — focused component not entirely hidden | AA |
| 2.4.12 ★ | Focus Not Obscured (Enhanced) — focused component not partially hidden | AAA |
| 2.4.13 ★ | Focus Appearance — focus indicators meet size (2px perimeter) and contrast (3:1) | AAA |
